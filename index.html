Women make up only 28% of the STEM workforce. This newsletter aims to change that.

The 28% is a monthly newsletter created by the women in STEM at PHS and JMHS. The goal of this project is to support girls/women in STEM and build community! Each month will feature relevant events, scholarship/program information, features on some cool women, and articles/artwork created by students. Subscribe below to receive the newsletter in your inbox each month!
